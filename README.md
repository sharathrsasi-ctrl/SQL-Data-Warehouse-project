# SQL-Data-Warehouse-project
Building a modern data warehouse using SQL, including ETL processes, Data modeling and Analytics.
# End-to-End Data Warehouse Project
This project follows the **Medallion Architecture** to transform raw data into business-ready insights.

### Architecture:
- **Bronze Layer**: Ingestion of raw CSV data into SQL Server.
- **Silver Layer**: Cleaning, deduplication, and schema standardization.
- **Gold Layer**: Star schema design with Fact and Dimension tables for reporting.

### Tools:
- SQL Server / Azure SQL / MY SQL
- Python (Pandas/SQLAlchemy)
- Tableau (for final visualization)
